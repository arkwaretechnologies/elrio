# **App Name**: El Rio - POS

## Core Features:

- Item Display: Display available menu items with their prices and stock levels.
- Cart Management: Allow users to add or remove items from the cart, adjusting quantities as needed.
- Total Calculation: Display a running total of the items in the cart.
- Discount Application: Enable users to apply discounts or promotions to the cart total.
- Menu Navigation: Allows user to view inventory, reports, settings, and user management
- Sign-out Functionality: Allows the users to Sign-out of the Point-of-Sale system

## Style Guidelines:

- Primary color: Light orange (#FFB74D) to create a warm and inviting atmosphere, reflecting the bakery's offerings.
- Background color: Very light yellow (#FFF9E6), creating a soft and clean backdrop that complements the main content.
- Accent color: Muted red-orange (#FF8A65) to highlight key actions and interactive elements.
- Font: 'PT Sans', a humanist sans-serif providing a modern, readable aesthetic for both headings and body text.
- Maintain a clean, intuitive layout with clear visual hierarchy to enhance user experience.
- Utilize simple, visually consistent icons to represent menu categories and actions, enhancing usability.
